# SaasMatchup — Core
*Shared across all SaasMatchup agents. Every agent reads this.*

---

## What it is

Three-sided marketplace for B2B SaaS stack decisions.

- **Side 1: Buyers** — get free AI-powered stack recommendations
- **Side 2: SaaS vendors** — get featured placement + qualified leads
- **Side 3: Consultants** — get matched to buyers who need help

**Live site:** https://saasmatchup.com
**Positioning:** "Stack architecture intelligence" — not a tool directory, not a review site. Tells you WHICH tools to use TOGETHER for YOUR company type.

---

## Primary moat

**AI Stack Builder** — 5-question quiz → Claude reasons over curated knowledge base → layered stack recommendation with rationale per tool. G2/Capterra cannot replicate multi-tool architecture recommendations. This ships in MVP, not Phase 3.

---

## Revenue model

1. **Affiliate commissions** — wired into every stack recommendation from launch (HubSpot, Monday, Klaviyo, Fivetran, Snowflake etc. all have programs)
2. **Vendor featured placement** — paid listings once traffic exists
3. **Consultant lead gen** — paid subscription once consultant directory has traction
4. **SEO compound growth** — hundreds of indexable pages: category pages, vendor pages, compare pages. "Best attribution tools 2026", "Fivetran vs Airbyte", "marketing data stack for agencies."

## SEO sprint plan (revised)

Original 14-sprint plan is superseded. The scaffold exists — now deepen and monetize.

| Priority | Sprint | What | Why |
|---|---|---|---|
| **Now** | Infra | sitemap.xml + robots.txt | 30 pages unindexed without it |
| Sprint A | Fill stubs | b2b-attribution, retail-media, analytics-bi, identity-resolution, affiliate-marketing | High-intent queries, some self-monetize |
| Sprint B | Compare pages | Triple Whale vs Northbeam, Rockerbox vs Measured, Fivetran vs Airbyte | Highest purchase-intent traffic, best affiliate conversion |
| Sprint C | Data Engineering | reverse-etl, ingestion-elt, transformation | Completes data layer |
| Sprint D | AI/ML | agents-orchestration, model-apis, observability | Category has index only |
| Sprint E | HRTech stubs | performance, people-analytics, talent-intelligence | Breadth play |
| Sprint F | /compare/ section | 10+ head-to-head compare pages | Compounding high-intent traffic |

**Success metric stays the same:** affiliate clicks, not traffic. First signal: 5+ affiliate clicks from attribution within 30 days of sitemap submission.
